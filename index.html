01 — 圧力とは

圧力とは何か

空気は何もないように見えますが、実は小さな粒（分子）がものすごい速さで飛び回っています。

この粒たちが壁にぶつかるたびに、壁を押す力が生まれます。この「たたく力の合計」が圧力です。

粒が多くぶつかるほど、または速くぶつかるほど、圧力は大きくなります。

グラフで整理する

体積（V）・温度（T）・圧力（P）の3つは深くつながっています。下のカードにカーソルを当てると、対応するグラフが強調されます。

// カードにカーソルを当ててみてください
[ P-V ]

体積が減ると圧力が増える

温度が一定なら、体積と圧力は反比例します。容器を縮めるほど圧力が上がります。

[ P-T ]

温度が上がると圧力が増える

体積が一定なら、温度と圧力は比例します。温めるほど圧力が上がります。

[ V-T ]

温度が上がると体積が増える

圧力が一定なら、温度と体積は比例します。温めるほど膨らみます。

[ N-P ]

粒が増えると圧力が増える

空気を追加で入れれば、ぶつかる粒が増えて圧力が上がります。

04 — REAL WORLD

身のまわりの圧力

圧力の知識は日常のあちこちで使われています。

[ 01 ]

自転車の空気入れ

ピストンで容積を縮めることで圧力を高め、タイヤへ送り込みます。

[ 02 ]

圧力鍋

密閉して加熱すると圧力が上がり、沸点が100℃以上になります。

[ 03 ]

スプレー缶

中身が高圧で充填されています。熱すると破裂の危険があります。

[ 04 ]

飛行機の耳ツン

気圧の急変化に耳の中の空気がついていけないことで起きます。

[ 05 ]

天気予報の低気圧

周囲より気圧が低い領域。空気が流れ込んで上昇し、雲ができます。

[ 06 ]

真空パック

袋の中の空気を抜くと、外側の大気圧が袋を押さえつけて密着します。

05 — QUIZ

理解度を確認する

Q密閉した容器に入った気体を温めると、圧力はどうなりますか？
// CORRECT : A温めると粒の速さが増し、壁へのぶつかりが激しくなります。そのため圧力が上がります。
Q真夏の車内に置きっぱなしのスプレー缶が危険な理由は何ですか？
// CORRECT : B密閉容器内の気体は温度が上がると圧力も上がります。容器が耐えきれずに破裂することがあります。
Q注射器のピストンを押し込んで空気を圧縮すると、何が起きますか？
// CORRECT : C粒の数は変わりません。空間が小さくなった分、壁にぶつかる頻度が増えて圧力として感じます。
Q画鋲が刺さりやすい理由を「圧力」の考え方で説明すると？
// CORRECT : A圧力 ＝ 力 ÷ 面積 です。面積が小さいほど同じ力でも圧力が大きくなります。画鋲の先は細いので圧力が集中して刺さります。
