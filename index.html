空気には、モノをぐっと押す力があります。飛行機に乗ったとき、耳がキーンと痛くなるのも、空気の力で押されているからなのです。この力のことを『圧力』と言います。今回は、そんな圧力を、一緒に見ていきましょう。

01 — 圧力とは

圧力とは何か

空気は何もないように見えますが、実は小さな粒（分子）がものすごい速さで飛び回っています。

この粒たちが壁にぶつかるたびに、壁を押す力が生まれます。この「たたく力の合計」が圧力です。

粒が多くぶつかるほど、または速くぶつかるほど、圧力は大きくなります。

02 — シミュレーション

ペットボトルに空気を入れたときの変化を見よう

▶ 使い方を見る
  1. 「ポンプを押す」をタップすると、ボトルに空気が1回分入ります。
  2. 圧力計・温度計の変化を観察しましょう。下の数値ボックスでも確認できます。
  3. 「グラフ ON」にすると、押した回数と圧力・温度の関係がグラフで表示されます。
  4. 「ふたを開ける」を押すと空気が抜けます。開けた瞬間の温度変化に注目！
  5. 「リセット」でいつでも最初からやり直せます。

03 — GRAPH VIEW

グラフで整理する

シミュレーションで起きていたことを2つのグラフで確認しましょう。

[ グラフ① ]　空気の量と圧力

空気を1回押すごとに圧力は一定量ずつ増える（比例関係）。大気圧（101.3 kPa）を切片として、グラフは右肩上がりの直線になります。

[ グラフ② ]　圧力と温度

空気を圧縮して入れると温度がわずかに上昇します。容器の体積が一定なら、圧力と温度は比例します（ゲイ＝リュサックの法則）。

// まとめ 空気の粒が「多い」ほど、また「速く動く（温度が高い）」ほど、壁への衝突が増えて圧力が上がります。
04 — REAL WORLD

身のまわりの圧力

圧力の知識は日常のあちこちで使われています。

※ カードをタップすると写真と解説が表示されます。

[ 01 ]

自転車の空気入れ

ピストンで空気を圧縮して圧力を高め、タイヤへ送り込みます。

[ 02 ]

スプレー缶

中身が高圧で充填されています。熱すると圧力が上がり破裂の危険があります。

[ 03 ]

天気予報の低気圧

周囲より気圧が低い領域。空気が流れ込んで上昇し、雲ができます。

[ 04 ]

真空パック

袋の中の空気を抜くと、外側の大気圧が袋を押さえつけて密着します。

05 — QUIZ

理解度を確認する

Q密閉した容器に入った気体を温めると、圧力はどうなりますか？
// CORRECT : A温めると粒の速さが増し、壁へのぶつかりが激しくなります。そのため圧力が上がります。
Q真夏の車内に置きっぱなしのスプレー缶が危険な理由は何ですか？
// CORRECT : B密閉容器内の気体は温度が上がると圧力も上がります。容器が耐えきれずに破裂することがあります。
Q注射器のピストンを押し込んで空気を圧縮すると、何が起きますか？
// CORRECT : C粒の数は変わりません。空間が小さくなった分、壁にぶつかる頻度が増えて圧力として感じます。
